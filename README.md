# Kollaborate
Nuestro proyecto de portafolio de titulo, una plataforma de colaboracion y aprendizaje academico





## Backend

Backend en desarrollo utilizando Node.js y Express.

## Pruebas basicas en Postman

se realizaron pruebas basicas utilizando postman para verificar el funcionamiento del backend

## Instalación
```bash
cd backend
npm install express cors bcrypt
```

## Ejecución
```bash  
node src/server.js
```
